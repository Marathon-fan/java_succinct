# how to write succinct Java code?

## 1  new an integer array

```java
	int x = 5;
	int y = 6;
	int z = 11;

	/////////////////////////////////
	////in this way, code is shorter
	int[] arr = new int[] {x, y, z};   
	System.out.println(arr);
	for (int cur : arr) {
		System.out.println(cur);
	}

```
